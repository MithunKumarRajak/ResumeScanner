# ResumeModel V4 — Production Implementation

## Overview

V4 is a multi-purpose resume AI platform that provides:

- **Classification**: Predict resume category from 48 job roles
- **Matching**: Score fit between resume and job description
- **Skill Extraction**: Extract and normalize technical skills
- **Quality Scoring**: Assess resume quality (stub for future enhancement)

## Architecture

```
Data Loading → Cleaning → TF-IDF Vectorization → Linear SVM (SGD + Calibration) → Predictions
                 ↓
           Skill List    Manifest JSON    Encoder
```

**Key Models:**

- **Classifier**: OneVsRestClassifier + CalibratedClassifierCV + SGDClassifier (linear SVM)
- **Features**: TF-IDF with 10,000 features, n-grams (1-3)
- **Test Accuracy**: 71.5% across 48 categories

## Setup

### 1. Install Dependencies

```bash
cd FullStackApp/backend
pip install -r requirements.txt
python -m spacy download en_core_web_sm
```

### 2. Train Model

From project root:

```bash
python FullStackApp/train/train_v4.py --data-dir Dataset --out-dir FullStackApp/v4
```

Options:

- `--use-transformer`: Enable sentence-transformers embeddings (requires more time/memory)
- `--embedder`: Specify transformer model (default: all-MiniLM-L6-v2)

Output: artifacts in `FullStackApp/v4/`

## API Usage

### Start Server

```bash
cd FullStackApp/backend
uvicorn app.main:app --reload --port 8000
```

### Endpoints

#### 1. Predict Category

```bash
curl -X POST http://localhost:8000/v4/predict-category \
  -H "Content-Type: application/json" \
  -d '{"resume_text": "Python engineer with 5 years AWS, Docker, Kubernetes"}'
```

Response:

```json
{
  "category": "Information Technology",
  "confidence": 0.82,
  "top5": [
    {"category": "Information Technology", "score": 0.82},
    {"category": "Backend Developer", "score": 0.15},
    ...
  ]
}
```

#### 2. Resume-Job Matching

```bash
curl -X POST http://localhost:8000/v4/match \
  -H "Content-Type: application/json" \
  -d '{
    "resume_text": "Python, AWS, Docker",
    "job_description": "Senior Python engineer with AWS and Docker experience required"
  }'
```

Response:

```json
{
  "score": 0.78,
  "overlap": ["python", "aws", "docker"]
}
```

#### 3. Skill Extraction

```bash
curl -X POST http://localhost:8000/v4/extract-skills \
  -H "Content-Type: application/json" \
  -d '{"resume_text": "Proficient in Python, Java, SQL, Docker, AWS"}'
```

Response:

```json
{
  "skills": ["python", "java", "sql", "docker", "aws"]
}
```

## Preprocessing Pipeline

**clean_text()**:

- Removes URLs, mentions, hashtags
- Strips non-alphabetic characters
- Collapses whitespace

**spacy_preprocess()**:

- Tokenizes and lemmatizes with spaCy
- Removes stopwords, punctuation, single-char tokens
- Returns lowercase token string

**Result**: Normalized, lemmatized text ready for TF-IDF vectorization

## Troubleshooting

**Model not loading**:

- Verify `FullStackApp/v4/` has all pkl files
- Check spaCy model is installed: `python -m spacy download en_core_web_sm`

**Slow preprocessing**:

- Normal for first run (spaCy initializes)
- Consider batching large datasets

**Low accuracy on custom data**:

- Train on your domain data using `train_v4.py --data-dir YOUR_DATA`
- Ensure consistent resume format and category labels
